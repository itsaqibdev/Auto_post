Method 1: Using Facebook Graph API Explorer (Recommended)

Step 1

Go to Facebook Developers

Open the Graph API Explorer:

üì± Open Graph API Explorer
Step 2

Select Your App

  • In the top dropdown, select your Facebook App (or create one)
  • Make sure you're using User Token (not App Token)
Step 3

Add Required Permissions

Click "Add a Permission" and enable these:

  • pages_show_list - View your Facebook Pages
  • pages_read_engagement - Read page data
  • instagram_basic - Access Instagram account
  • instagram_content_publish - Post to Instagram
  • business_management - Manage business assets
Step 4

Generate Token

  • Click "Generate Access Token"
  • Log in with Facebook and approve all permissions
  • Copy the access token that appears
Note: This is a short-lived token (expires in 1 hour). See Method 2 to get a long-lived token.
Step 5

Test in Admin Panel

  • Go to the Admin Panel
  • Paste the token in Instagram Access Token field
  • Click "üîç Find My Account ID"
  • It will automatically find and fill your Instagram Business Account ID

Method 2: Get Long-Lived Token (60 days)

Step 1

Get App Credentials

Go to your Facebook App settings:

üîß Facebook Apps
  • Click your app
  • Go to Settings > Basic
  • Copy App ID and App Secret
Step 2

Exchange Short Token for Long Token

Use this URL (replace YOUR_APP_ID, YOUR_APP_SECRET, and SHORT_TOKEN):

https://graph.facebook.com/v18.0/oauth/access_token?grant_type=fb_exchange_token&client_id=YOUR_APP_ID&client_secret=YOUR_APP_SECRET&fb_exchange_token=SHORT_TOKEN

Open this URL in your browser - you'll get a long-lived token that lasts 60 days.

Method 3: Check Instagram-Facebook Connection

Step 1

Verify Instagram is Business Account

  • Open Instagram app
  • Go to Settings > Account
  • Should show "Business" or "Creator"
  • If not, tap "Switch to Professional Account"
Step 2

Connect to Facebook Page

  • Go to Facebook Page (create one if you don't have)
  • Click Settings
  • Click Instagram in left sidebar
  • Click Connect Account
  • Log in to Instagram and authorize
‚úÖ After Connecting: Go back to the Admin Panel and click "Find My Account ID" - it should now work!
